VR Moda Masculina — Notas de Migração e Guia Rápido

Este documento resume o estado atual do repositório, alterações recentes e comandos úteis para executar o projeto localmente.

Resumo

  • Aplicação portada para React + Vite, com componentes em src/.
  • Estilos migrados para SASS (sintaxe indented) em src/styles/.
  • Imagens agora são importadas como módulos para otimização pelo Vite.
  • Testes E2E configurados com Playwright em e2e/.
  • Comentários explicativos em português foram adicionados a vários arquivos fonte (revise antes do commit final).

Executando localmente

powershell
npm install
# se for executar os testes E2E:
npx playwright install
npm run dev

# Em outro terminal, rode os testes E2E:
npm run test:e2e

Build de produção

powershell
npm run build

# Build compatível com comportamento "legacy" (carrinho oculto por padrão):
npm run build:legacy

Notas

  • Se o dev server (esbuild) falhar, rode npm install e verifique a versão do Node.js.
  • As imagens em src/components/Imagens/ são importadas como módulos nos componentes (Vite otimiza automaticamente).
  • Revise os comentários adicionados antes de commitar alterações finais.

Arquivos e pastas relevantes

  • src/ — código React, entrada em src/main.jsx e src/App.jsx.
  • src/components/ — Header, Carousel, Products, ProductCard, Cart, CheckoutForm, Footer, ScrollButton, icons.
  • src/styles/ — partials SASS e entry style.sass.
  • e2e/ — testes Playwright.
  • legacy/ — arquivos estáticos originais (referência).

Próximos passos recomendados

  1. Verifique alterações com git status e git diff.
  2. Execute npm run dev e valide visualmente as páginas principais.
  3. Opcional: execute os testes E2E com npm run test:e2e.
  4. Faça o commit com mensagem clara quando estiver satisfeito.
