VR Moda Masculina — Migration Notes

This document summarizes the migration to Vite + React and the current project layout.

What changed

  • Project migrated to Vite + React with componentized source under src/.
  • Styles converted to indented Sass (.sass) and organized in src/styles/.
  • Sass now uses @use with a shared _variables.sass file.
  • CSS Modules are used for per-component scoping (example: _product-card.module.sass).
  • Playwright E2E tests included under e2e/. Run with npm run test:e2e after installing browsers (npx playwright install).
  • Original static site files moved to legacy/ for reference (including legacy/index.original.html).

How to run

npm install
npm run dev  # dev server on port 5173
npm run build
npm run preview

# E2E tests:
npx playwright install
npm run test:e2e

Notes

  • The legacy/ folder contains the original static HTML/CSS/JS preserved for reference.
  • The visual layout should now be driven by React components and Sass partials; if you still see legacy markup, ensure the root index.html contains only the React mountpoint.
