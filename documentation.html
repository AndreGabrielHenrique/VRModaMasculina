VR Moda Masculina — Migration Notes

This document summarizes the migration to Vite + React and the current project layout.

What changed

  • Project migrated to Vite + React with componentized source under src/.
  • Styles converted to indented Sass (.sass) and organized in src/styles/.
  • Sass now uses @use with a shared _variables.sass file.
  • CSS Modules are used for per-component scoping (example: _product-card.module.sass).
  • Playwright E2E tests included under e2e/. Run with npm run test:e2e after installing browsers (npx playwright install).
  • Original static site files moved to legacy/ for reference (including legacy/index.original.html).

How to run

npm install
npm run dev  # dev server on port 5173
npm run build
npm run preview

# E2E tests:
npx playwright install
npm run test:e2e

VR Moda Masculina — Notas de Migração e Uso

        
Resumo do estado atual do repositório, instruções de desenvolvimento, build e execução dos testes.


        
Resumo

        
    
          
  • Aplicação portada para React + Vite, com código organizado em componentes dentro de src/.
    • 
          
  • Estilos migrados para SASS (sintaxe indented) e divididos em partials em src/styles/.
    • 
          
  • Testes E2E com Playwright em e2e/ — instale navegadores com npx playwright install.
    • 
          
  • Versão estática original preservada em legacy/ para referência.
    • 
        


        
Como executar localmente

        
powershell
    npm install
    npx playwright install   # apenas se for rodar os testes E2E
    npm run dev

    # Em outra janela, para testes E2E:
    npm run test:e2e
      


        
Build de produção

        
powershell
    npm run build

    # Build que reproduz comportamento 'legacy' (carrinho oculto por padrão):
    npm run build:legacy
      


        
Notas importantes

        
    
          
  • Se ocorrerem erros no dev server (esbuild), execute `npm install` e verifique a versão do Node. Mensagens de erro no terminal costumam indicar dependências faltantes.
    • 
          
  • Algumas alterações de código e estilo foram feitas e posteriormente editadas; revise as diferenças locais com git status e git diff antes de commitar.
    • 
          
  • Comentários explicativos em português foram adicionados a vários arquivos fonte — valide o conteúdo e remova/edite conforme sua preferência antes do commit final.
    • 
        


        
Arquivos e pastas de interesse

        
    
          
  • src/ — código React, entry em src/main.jsx e src/App.jsx.
    • 
          
  • src/components/ — Header, Carousel, Products, ProductCard, Cart, CheckoutForm, Footer, ScrollButton, icons.
    • 
          
  • src/styles/ — partials SASS e entry style.sass.

  • e2e/ — testes Playwright (homepage, cart interactions, visual snapshots).
