Project Documentation — VR Moda Masculina (migrated)

This HTML document contains a short overview of the migration to Vite + React and instructions you can keep locally.

How the migration was done

  • Created a Vite project with React and Sass.
  • Added `src/` with React components: `App`, `Header`, `Carousel`, `Products`, `ProductCard`, `Cart`, `Footer`.
  • Preserved original `style.css` and `script.js` in the repository root for reference.
  • Added `src/styles/style.sass` which imports the original `style.css` to keep current styling while enabling Sass features.

How to run

npm install
npm run dev

Notes

  • Images are referenced from `/Imagens/...` — keep that folder at the repository root.
  • If you want the original static site preserved, open `index.original.html` in a browser.
  • Further improvements: convert `style.css` to indented `.sass` partials and move component-specific styles into dedicated files.

If you want, I can proceed converting all CSS to idiomatic SASS and split styles per component.
