Project Documentation — VR Moda Masculina (migrated)

This HTML document contains a short overview of the migration to Vite + React and instructions you can keep locally.

How the migration was done

  • Created a Vite project with React and Sass (indented `.sass` syntax, not SCSS).
  • Added `src/` with React components: `App`, `Header`, `Carousel`, `Products`, `ProductCard`, `Cart`, `CheckoutForm`, `ScrollButton`, `Footer`.
  • Created 5 reusable icon components: `IconHeart`, `IconCart`, `IconSearch`, `IconUser`, `IconTrash`.
  • Ported all vanilla JS logic to React hooks and state management (cart, form validation, carousel auto-advance).
  • Converted all CSS to componentized SASS partials: `_header.sass`, `_carousel.sass`, `_products.sass`, `_cart.sass`, `_forms.sass`, `_footer.sass`, `_scroll-button.sass`, `_modal.sass`, `_media-queries.sass`.
  • Original files preserved in `legacy/` folder (`legacy/script.js` and `legacy/style.css`) for reference.
  • SASS Conversion 100% Complete — All styles migrated; CSS output reduced by 50% (~6.3 kB minified).

How to run

npm install
npm run dev

Notes

  • Images are referenced from `/Imagens/...` — keep that folder at the repository root.
  • All responsive design is maintained via `_media-queries.sass` (tablet 614-833px, mobile <613px).
  • Accessibility features: all icons have `aria-label`, form fields have `htmlFor` linking, interactive buttons are semantic.
  • The original static HTML structure is preserved as `index.original.html` in the root; run the React app with `npm run dev`.
  • Original vanilla JS (`legacy/script.js`) and CSS (`legacy/style.css`) are preserved in `legacy/` folder for future reference.
